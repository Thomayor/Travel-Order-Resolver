======================================================================
NORMALIZATION STEPS APPLIED
======================================================================

1. Hyphen Normalization
   - Converted en dash (–) → standard hyphen (-)
   - Converted em dash (—) → standard hyphen (-)
   - Converted minus sign (−) → standard hyphen (-)

2. Accent Removal
   - Removed French accents (é, è, à, ê, etc.)
   - Preserved base characters
   - Example: "Château" → "chateau"

3. Case Normalization
   - Converted all text to lowercase
   - Example: "PARIS" → "paris"

4. Special Character Removal
   - Removed non-alphanumeric characters
   - Preserved: spaces, hyphens, apostrophes
   - Example: "Gare (Paris)" → "gare paris"

5. Whitespace Cleanup
   - Collapsed multiple spaces to single space
   - Removed leading/trailing whitespace

6. Searchable Version Creation
   - Removed all spaces and hyphens
   - Example: "Aix-en-Provence" → "aixenprovence"

======================================================================
CITY EXTRACTION LOGIC
======================================================================

Strategy: Multi-level fallback with major city detection

1. Major City Detection (40 cities)
   - Paris, Lyon, Marseille, Lille, Bordeaux, Toulouse, Nice, etc.
   - If station name starts with major city name → extract city
   - Example: "Paris Austerlitz" → city = "Paris" ✓

2. Space-Dash-Space Separator
   - Pattern: "Station - Other Info"
   - Example: "Aguilcourt - Variscourt" → city = "Aguilcourt"

3. Hyphen Separator (compound names)
   - Pattern: "City-Suffix"
   - Example: "Port-Boulet" → city = "Port"
   - Example: "Aix-en-Provence" → city = "Aix"

4. Fallback: Full Station Name
   - For single-word stations without separators
   - Example: "Abbeville" → city = "Abbeville"
